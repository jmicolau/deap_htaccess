 
DirectoryIndex index.php
RewriteEngine on
RewriteBase /

RewriteCond %{REQUEST_FILENAME} !-f
RewriteCond %{REQUEST_FILENAME} !-d
RewriteRule ^(.*) index.php [L]

means that if the file with the specified name in the browser doesn’t exist, or the directory in the browser doesn’t exist then procede to the rewrite rule below

I know is that if you use RewriteRule ^(.*) index.php, it will redirect everything entered from http://www.sample-domain.com 255 to index.php so that you can parse the URI
